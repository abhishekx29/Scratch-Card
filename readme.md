# Scratch Card Project

A simple scratch card application built with JavaScript.

## Features

- Interactive scratch card effect
- Customizable card images and overlay
- Responsive design

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/abhishekx29/scratch-card.git
   ```
2. Open `index.html` in your browser.

## Usage

- Scratch the card using your mouse or touch to reveal the hidden image or message.

## Technologies

- JavaScript
- HTML5 Canvas
- CSS

## Customization

- Replace the overlay and reveal images in the `images` folder.
- Adjust scratch sensitivity in the JavaScript file.
